индивидуальное обучение основам web программирования в Харькове.

Обучение Go SQL HTML JS для школьников и взрослых.

Покажу как просто находить недостающюю информацию в интернете, работать с базой данных, писать код, импортировать данные, работать с хостингом и еще много всего.

Вы сделаете простое но эффективное веб приложение, в котором я покажу как решать задачи:
- покупки домена и его настройка
- подключение своего реального сайта с современным протоколом http/2 по SSL
- поиск в интернете и подбор html фреймворка для сайта
- вставка нужных форм и html элементов
- проектирование расположение меню, страниц и элементов сайта
- проектирование таблиц в БД
- выборка SQL для извлечения нужных данных на страницах
- Go, JS программирование логики работы сайта
